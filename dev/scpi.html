SCPI Interface

For communication betten the server and the client an SCPI with custom commands is used. The following table gives an overview of availalbe commands:

DIO

CommandArgumentsDescriptionExample
RP:DIO:DIRidentifier of pin, direction (IN/OUT)Set the direction of the DIORP:DIO:DIR DIO7_P,IN
RP:DIOidentifier of pin, value (0/1)Set the output of the DIORP:DIO DIO7_P,1
RP:DIO?identifier of pinGet the input of the DIORP:DIO? DIO7_P

ADC Configuration

CommandArgumentsDescriptionModeExample
RP:ADC:DECimationdecimation value [8, ..., n]Set the decimation factor of the sampling rateCRP:ADC:DEC 8
RP:ADC:DECimation?Return the decimation factorC, M, T
RP:TRIGger:MODeserver mode (CONFIGURATION, MEASUREMENT)Set the server modeCRP:TRIG:MOD CONFIGURATION
RP:TRIGger:MODe?Return the server modeC, M, TRP:TRIG:MOD?
